What is CatSystem?

CatSystem is a high performance development kit based on Swoole. It supports Http API Framework, RPC-based Micro Service Framework, Distributed Service Management Center, Service Management Dashboard and so on. And all of them are managed by Composer so that developers can easily use them in project.
